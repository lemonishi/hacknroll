# HushLight
A noise lamp system consisting of a lamp built with a Raspberry Pico Pi W and an analog sound sensor, and a web application which acts as a hub to control all active lamps.

## Features
- Lamps designed to blink rapidly upon detection of loud noise
- Drag and drop capabilities on the web application for flexible lamp positioning
- A log to keep track of areas where loud noise has been present for an extended period of time
- Handles locations or buildings with multiple floors

## Tech Stack
- React
- Flask
- Raspberry Pico Pi W (Micropython)
